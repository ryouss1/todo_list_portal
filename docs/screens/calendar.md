# Calendar画面 (`/calendar`)

> 本ドキュメントは [spec_function.md](../spec_function.md) から分割された画面仕様です。

## 表示機能

- FullCalendar ベースの月/週/日カレンダー表示
- サイドバーにユーザー・ソースフィルタ
- イベントの色分け表示（ユーザー設定のデフォルト色または個別色）
- 会議室予約の空き状況確認

## 操作機能

- **イベント作成**: モーダルダイアログで入力（タイトル、日時、場所、会議室、繰り返し、参加者）
- **イベント編集**: モーダルダイアログで修正（繰り返しイベントは変更スコープを選択）
- **イベント削除**: 確認ダイアログ後に削除（繰り返しイベントは削除スコープを選択）
- **参加者応答**: accept/decline/tentative で応答
- **カレンダー設定**: デフォルト表示・色・リマインダー等のユーザー個別設定
- **グループフィルタ**: サイドバーのグループドロップダウンでグループ単位のイベント表示を切替。グループ選択で該当メンバーのみチェックされ、イベントが再取得される

## カレンダーセル選択時の動作

| ビュー | クリック / ドラッグ箇所 | モーダル開始状態 |
|--------|----------------------|----------------|
| 月ビュー | 日付セル | 終日チェック OFF・開始 09:00 / 終了 10:00（時間入力可） |
| 週ビュー | 終日行（上部） | 終日チェック ON（時間入力なし） |
| 週ビュー | 時刻エリア | 終日チェック OFF・ドラッグ選択した時間範囲 |
| 日ビュー | 終日行（上部） | 終日チェック ON（時間入力なし） |
| 日ビュー | 時刻エリア | 終日チェック OFF・ドラッグ選択した時間範囲 |
