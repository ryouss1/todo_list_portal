# Alerts画面 (`/alerts`)

> 本ドキュメントは [spec_function.md](../spec_function.md) から分割された画面仕様です。

## 表示機能

- アラートを一覧表示（重要度バッジ、ソース、作成日時）
- アクティブ/全件のフィルタ切り替え
- 確認済み/未確認のステータス表示
- フィルタバー（アクティブフィルタとアラート一覧の間に配置）:
  - **重要度フィルタ**: ボタングループ（All / Info / Warning / Critical）
  - **ソースフィルタ**: ドロップダウン（アラート一覧から重複なしのソース名を自動抽出）
  - **キーワード検索**: タイトル・メッセージの部分一致検索（300msデバウンス）
- フィルタはクライアントサイドで即時適用（サーバー再取得なし）
- フィルタ条件はAND結合（重要度 AND ソース AND キーワード すべてに一致するアラートのみ表示）

## 操作機能

- **確認**: アラートを確認済みにマーク
- **非活性化**: アラートを非アクティブに変更
- **削除**: 管理者のみ削除可能

| 項目 | ファイル |
